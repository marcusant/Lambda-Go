ShareX Setup Guide
copying lambda config
Use CTRL+C to copy the contents of the following text area to the clipboard. You will need this in your clipboard for ShareX to recognize it.
creating a custom host
From the main ShareX window, click Destinations from the left-hand tab, and then click "Destination Settings." On the left-hand pane, scroll all the way down to "Custom uploaders" and select that. In the form that appears, click Import, and then From clipboard. You'll see a new option added in the list above the Import button called "Lambda". Your API key should be loaded in the table to the right. Lambda is now installed.
setting lambda as default
With Lambda installed, select Lambda from the drop-down menus for Image uploader and File uploader. You can click the Test buttons for either of them to ensure that it is working. These tests will upload the ShareX logo.
With Lambda now set as default, you can now upload freely using your assigned hotkeys. Enjoy!

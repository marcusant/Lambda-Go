Lambda Setup Guide (Linux)
support
The Lambda program is only tested on Arch Linux. It should work on other distributions, but you're on your own there.
getting lambda
Because I already package dependencies, the install process for Lambda is pretty simple.
Just download Lambda from the official download link and extract it either using your favorite archiving tool or with the command "tar -xzvf lambda-v*.tar.gz".
running lambda
Lambda doesn't have an installer, you just run it. In order to run Lambda, just double click it or run it with a terminal. Lambda will show up in the system tray (we currently have no plans to support window managers without a system tray).

If you are unable to run it because of a permissions issue, run "chmod 755 lambda" from the directory that lambda is in.
